Optical Character Recognition (OCR) Web Application

About the Project

A web-based application for performing Optical Character Recognition (OCR) using EasyOCR and Flask. The application allows users to upload images containing text, processes the images using a pre-trained OCR model, and extracts the text content, which is displayed on the web interface.

Key Features:

  • User-Friendly Interface: A simple interface for uploading images in various formats (e.g., PNG, JPG, JPEG, GIF).
  • OCR Processing: Integrated EasyOCR to perform text extraction from images with support for English language recognition.
  • File Management: Implemented secure file upload and storage mechanisms using Flaskâ€™s configuration.
  • Static Content Rendering: Real-time display of uploaded images alongside the extracted text.

Technologies Used:
Backend: Python, Flask, EasyOCR
Frontend: HTML, CSS
Libraries/Tools: Werkzeug, Secure Filename Handling
