Medical Chatbot using Llama2

About the Project

An AI-powered medical chatbot using the Llama2 model to assist with medical inquiries, providing accurate and reliable information.

➢ Technologies Used:

  • Python for scripting and automation
  • LangChain for integrating and managing multiple language models
  • FAISS for efficient vector storage and retrieval
  • PyTorch and Transformers for deep learning model implementation
  • Chainlit for creating interactive and dynamic prompts

➢ Implemented document loaders (PyPDFLoader, DirectoryLoader) to process medical documents and data

➢ Developed custom prompt templates to enhance the chatbot's response accuracy and reliability

➢ Utilized HuggingFaceEmbeddings for advanced natural language processing tasks

➢ Integrated CTTransformers and RetrievalQA for optimizing the chatbot's information retrieval capabilities

➢ Code: https://github.com/deadredraccoon21/Medical-chatbot-using-Llama2
