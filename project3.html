House Price Prediction Web App using Neural Networks

About the Project

➢ Developed a web application to predict house prices using a neural network model based on various input features.

Technologies Used:

  • React for building the frontend
  • Flask for creating the backend API
  • Python and TensorFlow/Keras for neural network model implementation
  • Axios for handling API requests between frontend and backend

➢ Implemented a responsive, user-friendly form for input collection on the frontend

➢ Designed and integrated a neural network model to process input data and predict house prices

➢ Styled the application with CSS to provide a visually appealing and functional interface
