Blockchain Based Cheating Prevention System In Fair Shop

About the Project

➢ Final year B.E. project: The objective of this project is to create a blockchain technology-based prototype that will act as an alternative to the traditional ration distribution system, which is prone to fraud activities and corruption.

➢ This is a simple end-to-end web application to replace the current food supply scheme using blockchain technology. It includes features and functionalities such as user registration, food distribution, and inquiry.

➢ The project uses Node.js, Ethers.js, ejs, and Solidity.

➢ Code: https://github.com/deadredraccoon21/Blockchain-based-cheating-prevention-system-in-fair-price-shop
